# Multiplication Table Generator 

A Python program that generates and displays the multiplication table for a given number from 1 to 10.

## Features
- Accepts any integer number as input
- Generates complete multiplication table from 1 to 10
- Displays each multiplication operation in a clear format
- Uses formatted output for clean table presentation
- Demonstrates basic arithmetic operations in a loop

## Concepts Used
- Integer input handling and type conversion
- For loops with `range()` function
- f-string formatting
- Arithmetic operations (multiplication)
- Iteration through a fixed sequence

## How to Run
```bash
python multiplication_table.py

# FizzBuzz Variant with Emoji 

A Python program that prints numbers from 1 to a given limit with special markers for multiples of 3 and 5.

## Features
- Accepts an integer input within range 1-20
- Validates input range before processing
- Prints numbers from 1 to the input number
- Marks multiples of 3 with ✨, multiples of 5 with ⭐, and both with ✨⭐
- Demonstrates conditional logic within a loop

## Concepts Used
- Integer input handling and range validation
- For loops with `range()` function
- Nested conditional statements (`if-elif-else`)
- Modulo operator (`%`) for divisibility checking
- Logical operator (`and`) for compound conditions
- f-string formatting with emoji

## How to Run
```bash
python fizzbuzz_emoji.py

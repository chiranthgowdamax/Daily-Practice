# Password Authentication System 

A Python program that repeatedly prompts for a password until the correct secret code is entered.

## Features
- Continuously prompts for password input until correct
- Provides immediate feedback on incorrect attempts
- Uses a predefined secret password for validation
- Grants access only when correct password is entered
- Demonstrates basic while-loop authentication

## Concepts Used
- String input handling
- While loops with condition checking
- String comparison
- Conditional flow control
- Basic authentication logic

## How to Run
```bash
python password_auth.py

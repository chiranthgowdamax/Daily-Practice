# Rectangle Pattern Generator 

A Python program that generates a rectangular pattern of asterisks based on user-specified dimensions.

## Features
- Accepts number of rows and columns as integer inputs
- Creates a rectangular grid of asterisks using nested loops
- Uses `end=""` parameter to print stars horizontally
- Properly formats output with line breaks after each row
- Demonstrates nested loop patterns for grid creation

## Concepts Used
- Integer input handling
- Nested for loops
- `range()` function for iteration control
- `print()` function with `end` parameter for horizontal output
- Basic pattern generation algorithms
- Grid-based output formatting

python main.py

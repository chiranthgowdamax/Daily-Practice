# Movie Ticket Bill

A simple Python program that calculates the total cost of movie tickets based on user input.

## Features
- Takes movie name as input
- Takes ticket price and number of tickets
- Calculates and prints the total cost with a message

## Concepts Used
- User input (`input()`)
- Typecasting (`int` or `float`)
- Variables
- Arithmetic multiplication
- Print statements

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

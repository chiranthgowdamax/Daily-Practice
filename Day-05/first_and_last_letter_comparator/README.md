# First and Last Letter Comparator 

A Python program that compares the first and last letters of a word to check if they are the same.

## Features
- Accepts any word as string input
- Converts input to lowercase for case-insensitive comparison
- Extracts first and last letters using string indexing
- Compares letters and provides clear comparison result
- Handles both single-letter and multi-letter words

## Concepts Used
- String input handling
- String indexing (positive and negative indices)
- `.lower()` method for case normalization
- Conditional statements (`if-else`)
- String comparison
- Variable assignment

## How to Run
```bash
python letter_comparator.py

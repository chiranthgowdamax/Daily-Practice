# Name Letter Extractor 

A Python program that extracts and displays the first and last letters of a given name using string indexing.

## Features
- Accepts any name as string input
- Extracts the first character using index `[0]`
- Extracts the last character using negative indexing or length calculation
- Displays both letters in a formatted output

## Concepts Used
- String input handling
- String indexing (positive and negative indices)
- `len()` function for string length
- f-string formatting
- Basic string manipulation

## How to Run
```bash
python name_letter_extractor.py

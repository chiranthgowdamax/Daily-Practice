# Middle Letter Finder 

A Python program that finds and displays the middle letter of a word with an odd number of letters.

## Features
- Accepts a word as string input (assumes odd length)
- Calculates the middle position using integer division
- Extracts the middle character using string indexing
- Displays the middle letter in a formatted output

## Concepts Used
- String input handling
- String indexing
- Integer division (`//` operator)
- `len()` function for string length
- f-string formatting
- Basic mathematical operations

## How to Run
```bash
python middle_letter_f

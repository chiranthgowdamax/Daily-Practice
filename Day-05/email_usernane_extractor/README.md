# Email Username Extractor 

A Python program that extracts the username part from an email address by finding the position of the '@' symbol.

## Features
- Accepts an email address as input
- Finds the position of the '@' symbol using string searching
- Extracts the username (characters before '@') using slicing
- Displays the extracted username in a formatted output

## Concepts Used
- String input handling
- `.find()` method for character searching
- String slicing with indices
- f-string formatting
- Basic string manipulation

## How to Run
```bash
python username_extractor.py

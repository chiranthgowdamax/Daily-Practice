# Title Formatter 

A Python program that centers a title text within a fixed width and adds decorative formatting.

## Features
- Accepts any title text as string input
- Centers the text within a specified width (30 characters)
- Adds decorative markers (**) on both sides of the formatted title
- Creates visually balanced, centered headers
- Demonstrates combination of text alignment and decoration

## Concepts Used
- String input handling
- f-string formatting with width and alignment specifiers
- Text centering (`^` for center alignment)
- Fixed-width formatting
- String decoration and combination

## How to Run
```bash
python title_formatter.py

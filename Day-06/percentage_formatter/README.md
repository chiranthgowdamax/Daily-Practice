# Progress Formatter 

A Python program that formats a percentage value as a right-aligned progress indicator.

## Features
- Accepts a percentage value as integer input (0-100)
- Right-aligns the number within a fixed width of 3 characters
- Appends the percentage symbol to the formatted number
- Creates clean, aligned progress display suitable for reports
- Ensures consistent formatting regardless of number size

## Concepts Used
- Integer input handling and type conversion
- f-string formatting with width and alignment specifiers
- Right alignment (`>` for right alignment)
- Fixed-width number formatting
- String concatenation in formatted output

## How to Run
```bash
python progress_formatter.py

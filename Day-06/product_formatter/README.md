# Product Formatter 

A Python program that formats product information with proper price formatting using f-strings.

## Features
- Accepts product name as string input
- Accepts product price as floating-point number
- Formats price to display with exactly two decimal places
- Combines product information into a clean, formatted string
- Uses f-string formatting for precise output control

## Concepts Used
- String input handling
- Float type conversion
- f-string formatting with precision specifiers
- Variable formatting (`:.2f` for two decimal places)
- String concatenation in output

## How to Run
```bash
python product_formatter.py

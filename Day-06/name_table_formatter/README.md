# Name Table Formatter 

A Python program that formats multiple names into a clean, aligned table structure using f-string formatting.

## Features
- Accepts three names as separate string inputs
- Aligns all names to the left with consistent width
- Creates a simple table structure with vertical separators
- Maintains uniform column width regardless of name length
- Demonstrates text alignment in formatted output

## Concepts Used
- String input handling
- f-string formatting with width and alignment specifiers
- Text alignment (`<` for left alignment)
- Fixed-width formatting
- Basic table structure creation

## How to Run
```bash
python table_formatter.py

# BMI Calculator with Categories ⚖️

A Python program that calculates Body Mass Index (BMI) from weight and height inputs and classifies the result into health categories.

## Features
- Accepts weight (kg) and height (m) as user input
- Calculates BMI using the standard formula: weight / (height²)
- Rounds BMI to one decimal place
- Classifies BMI into four health categories
- Validates height input to prevent division by zero

## Concepts Used
- Float type conversion
- Mathematical operations and exponentiation
- Conditional statements (`if-elif-else`)
- String formatting with f-strings
- Input validation
- Rounding numerical values

## How to Run
```bash
python bmi_calculator.py

A Python program that converts a numerical test score into a corresponding letter grade according to a standard grading scale.

- Handles invalid input scores outside the 0-100 range
- Uses a single print statement for output

- Conditional statements (`if-elif-else`)
- Logical and comparison operators
- Variable assignment and storage
- Input validation
- Integer type conversion

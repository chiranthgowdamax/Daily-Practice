# Input Value and Show Its Type

A simple Python program that takes a value from the user and displays the value along with its data type.

## Features
- Accepts user input
- Prints the entered value
- Shows the data type of the value using `type()`

## Concepts Used
- User input (`input()`)
- Print statements
- `type()` function
- Strings (default input type)

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

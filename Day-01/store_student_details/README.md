# Store Student Details

A simple Python program that stores basic student details using variables and prints them.

## Features
- Stores student name, age, and marks
- Uses direct value assignment (no user input)
- Prints each detail clearly

## Concepts Used
- Variables
- Data types (`string`, `int`, `float`)
- Print statements
- f-strings

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

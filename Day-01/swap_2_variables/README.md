# Swap Two Variables

A simple Python program that swaps the values of two variables and prints them before and after swapping.

## Features
- Stores two values in variables
- Swaps the values manually using a temporary variable
- Prints values before and after swapping

## Concepts Used
- Variables
- Data types (`int`, `string`, etc.)
- Print statements
- Basic value swapping using a temporary variable

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

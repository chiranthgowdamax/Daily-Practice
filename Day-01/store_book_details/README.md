# Store Book Details

A simple Python program that stores basic book details using variables and prints them.

## Features
- Stores book title, author, and number of pages
- Uses direct value assignment (no user input)
- Prints each detail clearly

## Concepts Used
- Variables
- Data types (`string`, `int`)
- Print statements
- f-strings

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

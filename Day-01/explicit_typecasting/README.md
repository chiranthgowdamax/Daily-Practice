# Explicit Typecasting

A Python program that takes a number as input, converts it to an integer using explicit typecasting, and displays the value along with its type.

## Features
- Accepts user input
- Converts input to integer using `int()`
- Prints the converted value
- Shows the data type of the converted value

## Concepts Used
- User input (`input()`)
- Explicit typecasting (`int()`)
- Print statements
- `type()` function

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

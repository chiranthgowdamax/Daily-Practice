# Pocket Money Tracker

A Python program that calculates the remaining pocket money after deducting expenses for snacks and travel.

## Features
- Takes starting pocket money as input
- Takes expenses for snacks and travel
- Calculates remaining money using assignment operators
- Rounds the final amount to 2 decimal places

## Concepts Used
- User input (`input()`)
- Typecasting (`float`)
- Assignment operator (`-=`)
- Arithmetic subtraction
- `round()` function
- f-strings for formatted output

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

# Age in Months Calculator

A Python program that converts a user's age in years into months and prints the result.

## Features
- Accepts age in years from the user
- Converts age to months using multiplication
- Prints the result using f-strings

## Concepts Used
- User input (`input()`)
- Typecasting (`int`)
- Arithmetic operators (`*`)
- Variables
- Print statements
- f-strings

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

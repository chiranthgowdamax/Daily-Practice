# Area of a Circle

A Python program that calculates the area of a circle based on the radius entered by the user.

## Features
- Accepts radius as input
- Calculates area using `pi * r * r`
- Uses `math.pi` for accurate Ï€ value
- Rounds the result to 2 decimal places

- Print statements

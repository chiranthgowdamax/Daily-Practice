# Simple Shape Identifier

A Python program that identifies whether a triangle is equilateral, isosceles, or scalene based on the lengths of its sides.

## Features
- Accepts three side lengths as input
- Compares side values to determine triangle type
- Prints the triangle classification clearly

## Concepts Used
- Variables
- User input (`input()`)
- Typecasting (`int`)
- Conditional statements (`if`, `elif`, `else`)
- Comparison operators
- f-strings (optional for prompts)

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

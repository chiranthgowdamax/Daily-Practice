# Simple Login Message

A Python program that takes a username and year of birth, calculates age, and displays a welcome message.

## Features
- Accepts username and year of birth
- Calculates age using subtraction
- Prints a personalized welcome message using f-strings

## Concepts Used
- User input (`input()`)
- Typecasting (`int`)
- Arithmetic operators (`-`)
- Variables
- Print statements
- f-strings

## How to Run
Assumption: This project is a single-file Python script named `main.py`. Current year is taken as 2026.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

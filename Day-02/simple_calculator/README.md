# Simple Calculator (Two Numbers)

A basic Python program that takes two numbers and prints their addition, subtraction, and multiplication.

## Features
- Accepts two numbers from the user
- Calculates addition, subtraction, and multiplication
- Prints results in a clear format

## Concepts Used
- User input (`input()`)
- Typecasting (`int` or `float`)
- Arithmetic operators (`+`, `-`, `*`)
- Print statements

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

# Adult or Minor Checker

A Python program that checks whether a person is an adult or a minor based on their age.

## Features
- Takes age as input
- Determines adult or minor using a simple condition
- Prints clear output based on the age

## Concepts Used
- User input (`input()`)
- Typecasting (`int`)
- Conditional statements (`if`, `else`)
- Boolean comparison operators

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

# Hypotenuse Calculator

A Python program that calculates the hypotenuse of a right-angled triangle using the Pythagoras theorem.

## Features
- Accepts base and perpendicular as input
- Calculates hypotenuse using `sqrt(base² + perpendicular²)`
- Uses `math.sqrt()` for square root calculation
- Rounds the result to 2 decimal places

## Concepts Used
- User input (`input()`)
- Typecasting (`float`)
- Arithmetic operators (`**`, `+`)
- `math` module (`math.sqrt()`)
- `round()` function
- Print statements
- f-strings

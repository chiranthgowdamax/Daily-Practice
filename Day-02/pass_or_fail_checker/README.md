# Pass or Fail Checker

A Python program that checks whether a student has passed or failed based on their marks.

## Features
- Takes student marks as input
- Checks pass or fail using a simple condition
- Prints result clearly as `Passed` or `Failed`

## Concepts Used
- User input (`input()`)
- Typecasting (`int`)
- Conditional statements (`if`, `else`)
- Boolean comparison operators

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

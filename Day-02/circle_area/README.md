# Circle Area Calculator

A Python program that calculates the area of a circle based on the radius entered by the user.

## Features
- Accepts radius as input
- Calculates area using the formula `area = 3.14 * r * r`
- Prints the calculated area

## Concepts Used
- User input (`input()`)
- Typecasting (`float`)
- Arithmetic operators (`*`)
- Variables
- Print statements

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

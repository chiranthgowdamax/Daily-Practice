# Student Result Card

A Python program that takes a student's name and marks for three subjects, calculates the total, and prints the result.

## Features
- Accepts student name and marks for three subjects
- Calculates total marks using addition
- Prints student name and total marks using f-strings

## Concepts Used
- User input (`input()`)
- Typecasting (`int` or `float`)
- Arithmetic operators (`+`)
- Variables
- Print statements
- f-strings

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py

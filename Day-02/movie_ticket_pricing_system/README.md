# Movie Ticket Pricing System

A Python program that calculates the final movie ticket price based on age, show time, and day type using multiple discount and surcharge rules.

## Features
- Uses a fixed base ticket price
- Applies age-based discounts
- Applies an additional time-based discount for morning shows
- Adds a weekend surcharge when applicable
- Outputs the final ticket price rounded to 2 decimal places

## Concepts Used
- Variables
- User input (`input()`)
- Typecasting (`int`)
- Arithmetic operators
- Conditional statements (`if / elif / else`)
- Comparison operators
- `round()` function
- f-strings

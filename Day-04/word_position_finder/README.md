A Python program that searches for a specific word within a message and returns its starting position using string methods.

## Features
- Accepts a message and search word as separate inputs
- Finds the starting position of the search word within the message
- Handles cases where the word is not found
- Demonstrates zero-based indexing in string positions

## Concepts Used
- String input handling
- `.find()` method for substring searching
- Conditional statements (`if-else`)
- Comparison with return value `-1` for not found cases
- String indexing and positions
- f-string formatting for output

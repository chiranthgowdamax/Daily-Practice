A Python program that censors specific words in a message and counts their occurrences using string methods.

## Features
- Accepts any text message as input
- Replaces target words with censorship symbols
- Counts occurrences of the target word in the original message
- Provides both censored message and occurrence count

## Concepts Used
- String input handling
- `.replace()` method for text substitution
- `.count()` method for substring counting
- Multiple variable assignment
- String manipulation and analysis
